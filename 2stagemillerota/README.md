# 2-Stage Miller OTA (Operational Transconductance Amplifier)

## Project Overview

This project covers the complete design and analysis of a 2-stage Miller OTA circuit with compensation techniques. The design successfully achieved all target specifications through systematic methodology and iterative refinement.

## Final Performance

| Parameter              | Target      | Achieved  |
| ---------------------- | ----------- | --------- |
| DC Gain                | ≥ 60 dB     | 77.6 dB   |
| UGB                    | ≥ 10 MHz    | 10.53 MHz |
| Phase Margin           | ≥ 60°       | 64.7°     |
| Power Consumption      | ≤ 100 µW    | 34.7 µW   |
| Positive Slew Rate     | ≥ 5 V/µs    | 6.5 V/µs  |
| Negative Slew Rate     | —           | 12.5 V/µs |
| Load Capacitance       | 1 pF        | 1 pF      |
| Supply Voltage         | 1.8 V       | 1.8 V     |
